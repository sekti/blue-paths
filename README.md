# Blue Paths

[Blue Prince](https://www.blueprincegame.com) is a rogue-like puzzle game by [Dogubomb](http://www.dogubomb.com). This repository contains the code for *Blue Paths* an interactive hint system for Blue Prince deployed as a [website on the author's github pages](https://sekti.github.io/blue-paths). On that page you will more information about this project.

## Technical Information

To deploy this project yourself, run
```
npm install
npm run build
npm run preview
```
Then open localhost:4173/blue-paths/ in your webbrowser (the port might vary).

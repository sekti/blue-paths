# Blue Paths

[Blue Prince](https://www.blueprincegame.com) is a rogue-like puzzle game by [Dogubomb](http://www.dogubomb.com). This repository contains the code for *Blue Paths* an interactive hint system for Blue Prince deployed as a [website on the author's github pages](https://sekti.github.io/blue-paths). On that page you will find more information about this project.

The project is not affiliated with or endorsed by Dogubomb.

## Technical Information

To test this project locally, run
```
npm install
npm run dev
```
Then open localhost:5173/blue-paths/ in your webbrowser (the port might vary).
